IP Security

IP banning, rules, and GeoIP filtering

{{navIcon "warning"}}

Security module is disabled

Enable it in data/config.json and restart the server: 

"security": {
  "enabled": true,
  "auto_ban_enabled": true,
  "auto_ban_threshold": 5,
  "auto_ban_window_minutes": 60,
  "auto_ban_duration_hours": 24,
  "geoip_enabled": false,
  "maxmind_license_key": "",
  "geoip_update_days": 7
}
🛡️

Auto-ban

Automatically ban IPs that exceed the failed login threshold. Default: 5 failures in 60 minutes → 24-hour ban.

📋

IP Rules

Whitelist trusted IPs, blacklist known bad actors, or lock down paths to specific IP ranges.

🌍

Country Filtering

Block or allow traffic by country using MaxMind GeoLite2 (free, requires license key, auto-updates weekly).

📊

Attempt History

View all failed login attempts with IP, username, and whether they triggered an auto-ban.
