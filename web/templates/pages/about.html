How it Works

This app demonstrates a Flask-style template inheritance pattern using Go's html/template package, paired with full user management, leveled logging, and a modular page system.

1

One base layout

base.html defines the full HTML shell: sidebar nav, header, footer, flash messages, and named blocks for title, content, and scripts.

2

Page templates override blocks

Each page starts with {{"{{"}}template "base" .{{"}}"}} then redefines only the blocks it needs.

3

Renderer pairs them at startup

The Renderer pre-parses base + each page into its own *template.Template. Executing "base" always picks up the correct page blocks — no blank pages.

4

Handlers are independent modules

Every page is a separate Go file implementing http.Handler. Register it in router.go and you're done.
